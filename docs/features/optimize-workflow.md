# Optimize Workflow Spec

## Loop Structure

1. Create a prompt
2. Run it
3. Collect output
4. Evaluate output
5. Find issues
6. Generate improved prompt
7. Save version
8. Repeat

## What The Workflow Must Capture

- original prompt
- output text
- target goal
- detected issues
- fix suggestions
- revised prompt
- version history

## Why This Matters

Without output feedback, optimization becomes guesswork.

This workflow makes improvement measurable and repeatable.
